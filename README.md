## 🌟 What is this?

This repository is a **complete, hands-on guide to R programming** built for university students who are just getting started. Every topic includes:

- 📝 **Plain-English notes** explaining the concept before any code
- 🖥️ **RStudio Console screenshots** showing exactly what to type
- ✅ **Expected output** so you know if your code is working
- 💡 **Tips and gotchas** to help you avoid common mistakes

## 🚀 How to Use This Repo

1. Read the **notes** at the top of each file — understand *why* before *how*
2. Open RStudio on your computer
3. Type the commands shown in the **Console screenshots**
4. Check your output matches the **expected result**
5. Try the **exercises** at the end of each section
