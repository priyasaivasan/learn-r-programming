## 🤔 What is R?

R is a **free, open-source programming language** designed specifically for **statistics, data analysis, and data visualisation**. Think of it as a very powerful calculator that can also draw graphs, run complex models, and handle enormous datasets — all with just a few lines of code.

Unlike general-purpose languages (like Python or Java), R was built *by statisticians, for statisticians*. This means it speaks the language of data natively.

---

## 🕰️ A Brief History

| Year | Milestone |
|------|-----------|
| 1976 | **S language** created at Bell Labs by John Chambers — R's ancestor |
| 1991 | **R created** by Ross Ihaka and Robert Gentleman at University of Auckland, NZ |
| 1995 | R released as **open-source** software |
| 1997 | **CRAN** (Comprehensive R Archive Network) launched — home of R packages |
| 2000 | **R 1.0.0** officially released |
| Today | Over **20,000+ packages** on CRAN; used by millions worldwide |

---

## ✨ Key Features of R

- 🆓 **Free and open-source** — no licence fees, ever
- 📦 **20,000+ packages** — tools for almost any analytical task
- 📊 **World-class graphics** — `ggplot2` produces publication-quality charts
- 🤝 **Huge community** — Stack Overflow, GitHub, CRAN, R-bloggers
- 🔗 **Connects to everything** — SQL databases, Excel, Python, cloud platforms
- 🧪 **Reproducible research** — R Markdown lets you mix code, text, and output

## 💡 R vs Python — Which Should You Learn?

> **Short answer for university students: learn both eventually, start with R if your course is stats-heavy.**

| | R | Python |
|--|---|--------|
| Strength | Statistics & visualisation | General programming & ML |
| Learning curve | Gentle for data tasks | More versatile but broader |
| Used in | Academia, biostatistics, finance | Industry, AI, web |
| Packages | `ggplot2`, `dplyr`, `tidyverse` | `pandas`, `scikit-learn`, `numpy` |
